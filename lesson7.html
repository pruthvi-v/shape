Position: static

Static is always positioned according to the normal flow of the page

Div element has position static

Position: relative

Is positioned relative to its normal position

Div element has position relative. shifting 30px to left from default position

Position: fixed

Is positioned relative to the viewport, which means it always stays in the same plays even if the page is scrolled.

Div element has position fixed.

Position: absolute

An element is positioned relative to the nearest positioned anscestor (instead of positioned relative to the viewport) like fixed.

Relative element
Div element has absolute relative

Position: sticky

Sticky element sticks to the top of the page (top:0)

Div element has position sticky
